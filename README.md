# ACSL #2 Junior Coding Challenge


### Language
JavaScript using Node.js v14.15.3 LTS

### Running
###### Install dependencies
```shell
$ npm install
```
###### Insert input
 1. Rename the `input.txt.example` file to `input.txt`.
 2. Insert data into the new file.

###### Running
```shell
$ node acsl.js
```
